Ecryption & Decryption


This Python code is a basic tool for Caesar cipher encryption and decryption. It presents a menu with options to encrypt, decrypt, or exit. The user provides a numeric key to shift characters in the input text, and the cipher function performs the transformation. After encryption or decryption, the result is displayed. The user can repeat the process or exit the program. It's a straightforward command-line tool for basic text encryption and decryption using a Caesar cipher.

Unique Matrix


This C++ code checks if a 4x4 matrix of 16 unique numbers input by the user forms a magic square. A magic square has equal sums in rows, columns, and diagonals. The user is prompted to enter values, and the code validates them. If it's a magic square, it displays the matrix with "Magic square!" or "Not a magic square!" if not.
